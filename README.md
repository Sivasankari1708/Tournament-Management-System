# Tournament Management System 🏆

A modern full-stack tournament registration platform built using React, TypeScript, Tailwind CSS, FastAPI, and Firebase Authentication.

---

# Features

## Frontend
- Modern responsive UI
- Tournament cards
- Sports images
- Registration forms
- Team registration
- OTP authentication
- Tournament rules display
- Dynamic fee display

## Backend
- FastAPI REST APIs
- Tournament data APIs
- Team registration APIs
- Firebase OTP integration
- JSON-based tournament management

---

# Tech Stack

## Frontend
- React
- TypeScript
- Tailwind CSS
- React Router DOM
- Axios

## Backend
- FastAPI
- Python
- Uvicorn

# Installation

## Frontend

```bash
cd frontend
npm install
npm run dev
```

## Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
